Virtual Machine Protection

CrypKey SDK and INSTANT products now feature effective protection for virtual machines. VMProtect by CrypKey is a new and innovative way to protect applications and documents from illegal usage by virtual machine duplication. VMProtect reads real hardware data from the host machine to detect if a virtual machine is running. Because other data such as drive serial number and motherboard serial number are easily replicated in a virtual machine, checking only for this information means a virtual machine can bypass copy protection and licensing. With VMProtect able to detect the presence of a virtual machine, your applications and documents are safe from this loophole.
